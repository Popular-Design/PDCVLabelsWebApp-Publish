# Contributing to PDCVLabelsWebApp

Thank you for your interest in contributing to PDCVLabelsWebApp!

## About This Repository

This is a **public publishing repository**. The actual development happens in a private repository, and changes are published here automatically.

## How to Report Issues

If you encounter any bugs or have feature requests:

1. **Search Existing Issues**: Check if your issue has already been reported
2. **Create a New Issue**: If not found, open a new issue with:
   - A clear and descriptive title
   - Detailed description of the issue or feature request
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Any relevant screenshots or logs

## Development Contributions

Since this is a publishing repository:
- Direct pull requests to this repository cannot be accepted
- Issues and bug reports opened here will be reviewed by maintainers
- Fixes and features will be implemented in the private source repository
- Changes will be published here once completed and tested

## Questions and Support

For questions about using PDCVLabelsWebApp:
- Check the README.md for basic usage information
- Review existing issues for common questions
- Open a new issue with the "question" label

## Code of Conduct

Please be respectful and constructive in all interactions. We aim to maintain a welcoming and inclusive community.

## Contact

For private inquiries or security concerns, please contact the repository maintainers directly.
